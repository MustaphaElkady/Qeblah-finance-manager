# Media Finance Manager

A Streamlit-based internal system for media companies to manage:

- Clients
- Packages
- Contracts
- Payments
- Expenses
- Profitability Reports

## Features
- Client management
- Fixed package management
- Contract tracking
- Payment tracking
- Expense tracking
- Dashboard summary
- Reports with filters
- CSV export
- Basic login

## Default Login
- Username: admin
- Password: admin123
